10/07/22 - I host Instances now!;


You can read can't you? It's right up there if you can't tell. I now host public instances for several different project. But what is an Instance? Well in short when you go onto a site, for example let's say Reddit alot of information is being logged, on the official Reddit website your IP address, User-agent, Browser, Operating System, any referal links, the device ID, pages visited, links clicked, search terms, and more are being collected by a company that deletes anything that does not live up to the "Wholesome 100" look that they want.

So let's say you want to use Reddit but don't want the Big Keanu Chungus Reeves company to know anything about you. Well then you are in luck. There are several projects such as Teddit and Libreddit that provides an "alternative front-end" to these sites and don't share you presonal information.
So I have some of these project and set them up so that anyone can use them. Of course these things cost money so if you like my instances why not donate? Yes I have a donation page no I don't care... I like money.

SearXNG;

SearXNG is a metasearch engine which basicly means it takes results from other search engines such as Google and DuckDuckGo and presents them to the user. On top of that it is also privacy-respecting, well as privacy-respecting as the search engines you have enabled are.

I think that metasearch engines are much better then normal search engines because some search engines do things much better than others so instead of having to go to different websites to get the link you want you can just use one site.

Repo: https://github.com/searxng/searxng
Link: https://searx.sevengames.xyz

Invidious;

Invidious is an open source privacy-respecting alternative front-end to YouTube. If you've read the previous instances descriptions then you know what this is. It allows users to view YouTube videos without going to the YouTube site. Invidious is better than regluar YouTube not because is not owned by Google but because it is a lightweight site that just works... well and because it's not owned by Google.
